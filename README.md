| Comando | O que faz? | Exemplo |
| :--- | :--- | :--- |
| **SELECT** | Seleciona colunas específicas | `SELECT nome FROM usuarios;` |
| **SELECT *** | Seleciona todas as colunas | `SELECT * FROM usuarios;` |
| **WHERE (Número)** | Filtra resultados numéricos | `SELECT * FROM usuarios WHERE idade > 18;` |
| **WHERE (Texto)** | Filtra resultados de texto | `SELECT nome FROM usuarios WHERE cidade = 'São Paulo';` |
